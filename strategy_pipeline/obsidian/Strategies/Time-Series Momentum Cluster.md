## Overview

Time-series momentum (TSM) is the academic formalization of trend following: if an asset's trailing return over a lookback period is positive, go long; if negative, go short. Chan provides the quantitative framework with results varying by lookback/hold period: TU (250/25) Sharpe 1.0, Brazilian Real (100/10) Sharpe 1.09, Copper (40/40) Sharpe 1.05, with 17-18% APR on commodities. Key finding: shorter lookbacks work better for commodities because supply-demand cycles are faster. Basso's volatility-normalized trend system is the risk management complement. Parker's Chesapeake Capital trades 30-65 markets using similar multi-asset momentum.

## Core Rules Consensus

1. **Positive trailing return = go long; negative = go short.** The simplest possible momentum signal. Chan demonstrates this works across asset classes.
2. **Shorter lookbacks work better for commodities.** Commodities (40 days optimal) vs bonds (250 days optimal). Supply-demand cycles drive the difference.
3. **Normalize position size by volatility.** Basso and Chan both adjust positions by current ATR/volatility. Higher vol = smaller position.
4. **Multi-asset diversification amplifies the edge.** Parker (30-65 markets) demonstrates that momentum across many uncorrelated markets produces smoother returns than any single market.
5. **Roll return adds signal for futures.** Chan documents that combining price momentum with roll-return signals improves Sharpe for futures strategies.

---

## Key Differences

- **Chan uses simple return-over-lookback as signal; traditional trend followers use MA crossovers or channel breakouts**
- **Chan's optimal lookback is shorter for commodities (40 days) vs bonds (250 days)**
- **Basso normalizes position size by volatility; Chan uses Kelly criterion**
- **Parker favors longer lookbacks (52-week moving average equivalent)**
