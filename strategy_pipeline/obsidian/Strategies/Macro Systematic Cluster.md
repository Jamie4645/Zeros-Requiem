## Overview

This cluster covers systematic approaches to multi-asset allocation and macro trading. Dalio's All Weather framework (4 quadrants: growth up/down × inflation up/down) provides the theoretical basis for multi-asset allocation. O'Shea's hypothesis-testing macro approach uses asymmetric trade structures (defined-risk options/CDS). Druckenmiller focuses on central bank policy as the dominant driver. Chan's cross-asset strategies (VX-ES, XLE-USO) exploit structural relationships between asset classes.

## Core Principles

1. **Asset allocation is the primary return driver.** Dalio: 100+ uncorrelated streams reduce portfolio volatility more than any single strategy improvement.
2. **4-quadrant framework for regime identification.** Growth up/down × Inflation up/down defines which assets perform. Gold thrives in "inflation up" regimes.
3. **Hypothesis testing with defined risk.** O'Shea: form a macro thesis, express it with asymmetric risk structures, scale if the thesis plays out.
4. **Central bank policy dominates medium-term moves.** Druckenmiller: the most powerful force in markets is central bank monetary policy.
5. **Cross-asset relationships provide diversification and hedging.** Chan: cointegrated cross-asset pairs (e.g., VIX-S&P) offer uncorrelated return streams.

## Key Differences

- **Dalio is fully systematic with extreme diversification (100+ streams)**
- **Druckenmiller is highly discretionary with concentrated bets**
- **O'Shea uses hypothesis testing with defined-risk implementation (options/CDS)**
- **Chan provides quantitative tools (cointegration, roll return) for systematic cross-asset strategies**
